abs (AutoLISP)

Returns the absolute value of a number

Supported Platforms: Windows, Mac OS, and Web

Signature 

(abs number)
number

Type: Integer or Real

A numeric value.

Return Values

Type: Integer or Real

The absolute value of the number argument.

Examples 

(abs 100)
100

(abs -100)
100

(abs -99.25)
99.25
