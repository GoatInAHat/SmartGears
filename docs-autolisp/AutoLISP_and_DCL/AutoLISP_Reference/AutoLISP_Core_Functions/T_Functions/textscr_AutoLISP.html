textscr (AutoLISP)

Switches focus from the drawing area to the text screen

Supported Platforms: Windows, Mac OS, and Web

Signature 

(textscr)

No arguments.

Return Values

Type: nil

Always returns nil .

Remarks

Note: This function is supported on Mac OS and Web, but does not affect the program.
