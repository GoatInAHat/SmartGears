reverse (AutoLISP)

Returns a copy of a list with its elements reversed

Supported Platforms: Windows, Mac OS, and Web

Signature 

(reverse lst)
lst

Type: List

A list of the items to reverse.

Return Values

Type: List

Values of the lst argument reversed.

Examples 

(reverse '((a) b c))
(C B (A))
