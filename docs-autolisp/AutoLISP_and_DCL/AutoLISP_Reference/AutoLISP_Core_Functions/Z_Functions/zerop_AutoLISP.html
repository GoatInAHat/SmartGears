zerop (AutoLISP)

Verifies that a number evaluates to zero

Supported Platforms: Windows, Mac OS, and Web

Signature 

(zerop number)
number

Type: Integer or Real

A numeric value.

Return Values

Type: T or nil

T if number evaluates to zero; otherwise nil .

Examples 

(zerop 0)
T

(zerop 0.0)
T

(zerop 0.0001)
nil
