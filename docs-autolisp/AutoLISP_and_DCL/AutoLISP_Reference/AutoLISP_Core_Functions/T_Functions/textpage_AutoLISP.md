# textpage (AutoLISP)

> Switches focus from the drawing area to the text screen

## Quick Reference

- Source: [https://help.autodesk.com/cloudhelp/2025/ENU/AutoCAD-AutoLISP-Reference/files/GUID-C5F1E0AF-CAE8-4338-8C27-5A4235D37145.htm](https://help.autodesk.com/cloudhelp/2025/ENU/AutoCAD-AutoLISP-Reference/files/GUID-C5F1E0AF-CAE8-4338-8C27-5A4235D37145.htm)

**Supported Platforms:**  Windows, Mac OS, and Web

## Signature

```lisp
(textpage)
```

No arguments.

## Return Values

**Type:**  nil

Always returns `nil`.

## Remarks

The `textpage`  function is equivalent to `textscr`.

Note:
 This function is supported on Mac OS and Web, but does not affect the program.
