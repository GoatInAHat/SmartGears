1- (decrement) (AutoLISP)

Decrements a number by 1

Supported Platforms: Windows, Mac OS, and Web

Signature 

(1- number)
number

Type: Integer or Real

A numeric value.

Return Values

Type: Integer or Real

The number argument, reduced by 1.

Examples 

(1- 5)
4

(1- -17.5)
-18.5
