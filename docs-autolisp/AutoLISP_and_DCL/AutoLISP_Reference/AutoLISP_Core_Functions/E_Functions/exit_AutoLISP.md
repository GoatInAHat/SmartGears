# exit (AutoLISP)

> Forces the current application to quit

## Signature

```lisp
(exit)
```

No arguments.

## Return Values

**Type:**  nil

If `exit`  is called, it outputs the error message quit/exit abort and returns to the AutoCAD Command prompt.
