float (AutoLISP)

Returns the conversion of a number into a real number

Supported Platforms: Windows, Mac OS, and Web

Signature 

(float number)
number

Type: Integer or Real

A numeric value.

Return Values

Type: Real

The real number derived from number .

Examples 

(float 3)
3.0

(float 3.75)
3.75
