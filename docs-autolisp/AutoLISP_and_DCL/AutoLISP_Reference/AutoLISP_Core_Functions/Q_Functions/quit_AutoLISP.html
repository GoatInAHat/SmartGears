quit (AutoLISP)

Forces the current application to quit

Supported Platforms: Windows, Mac OS, and Web

Signature 

(quit)

No arguments.

Return Values

Type: nil

If quit is called, it returns the error message quit/exit abort and returns to the AutoCAD Command prompt.
