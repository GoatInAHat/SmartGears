cos (AutoLISP)

Returns the cosine of an angle expressed in radians

Supported Platforms: Windows, Mac OS, and Web

Signature 

(cos ang)
ang

Type: Integer or Real

An angle, in radians.

Return Values

Type: Real

The cosine of ang , in radians.

Examples 

(cos 0.0)
1.0

(cos pi)
-1.0
