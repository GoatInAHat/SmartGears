itoa (AutoLISP)

Returns the conversion of an integer into a string

Supported Platforms: Windows, Mac OS, and Web

Signature 

(itoa int)
int

Type: Integer

A numeric value.

Return Values

Type: String

A string derived from int .

Examples 

(itoa 33)
"33"

(itoa -17)
"-17"
