# To Create and Open AutoLISP Source Code Files (AutoLISP)

> AutoLISP source code files can be created and edited using a plain text editor.

## Create a new source code file

- **Windows:** Click Start ![](../../../../../_assets/ac_menuaro-221ec883.gif)  Windows Accessories ![](../../../../../_assets/ac_menuaro-221ec883.gif)  Notepad.  In Notepad, click File ![](../../../../../_assets/ac_menuaro-221ec883.gif)  Save As.  In the Save As dialog box, click the Save As Type drop-down list and select All Files (*.*).  Specify a location for the file and enter a name in the File Name box. Add **.lsp**  after the file name and click Save.
- **Mac OS:** In Finder, click Go ![](../../../../../_assets/ac_menuaro-221ec883.gif)  Applications.  In the Applications window, double-click TextEdit.  In TextEdit, if you see the ruler and formatting bar along the top of the window, do the following: Click TextEdit ![](../../../../../_assets/ac_menuaro-221ec883.gif)  Preferences.  In Preferences, Format, click Plain Text. Click Close.  Click File ![](../../../../../_assets/ac_menuaro-221ec883.gif)  New.  Click File ![](../../../../../_assets/ac_menuaro-221ec883.gif)  Save As.  In the Save As dialog box, specify a location for the file and enter a name in the Save As box. Add **.lsp**  after the file name and click Save.  In the Extension dialog box, click Use .LSP.

## Open an existing source code file

- **Windows:** Click Start ![](../../../../../_assets/ac_menuaro-221ec883.gif)  Windows Accessories ![](../../../../../_assets/ac_menuaro-221ec883.gif)  Notepad.  In Notepad, click File ![](../../../../../_assets/ac_menuaro-221ec883.gif)  Open.  In the Open dialog box, click the Save As Type drop-down list and select All Files (*.*).  Browse to and select the LSP file to open. Click Open.
- **Mac OS:** In Finder, click Go ![](../../../../../_assets/ac_menuaro-221ec883.gif)  Applications.  In the Applications window, double-click TextEdit.  In TextEdit, click File ![](../../../../../_assets/ac_menuaro-221ec883.gif)  Open.  In the Open dialog box, browse to and select the LSP file to open. Click Open.
