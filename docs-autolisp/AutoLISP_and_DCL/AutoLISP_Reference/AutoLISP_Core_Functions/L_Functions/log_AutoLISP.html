log (AutoLISP)

Returns the natural log of a number as a real number

Supported Platforms: Windows, Mac OS, and Web

Signature 

(log num)
num

Type: Integer or Real

A positive number.

Return Values

Type: Real

A numeric value.

Examples 

(log 4.5)
1.50408

(log 1.22)
0.198851
