# terpri (AutoLISP)

> Prints a newline to the command line

## Quick Reference

- Source: [https://help.autodesk.com/cloudhelp/2025/ENU/AutoCAD-AutoLISP-Reference/files/GUID-16FB5411-7563-4FB9-AB59-A73457B092C5.htm](https://help.autodesk.com/cloudhelp/2025/ENU/AutoCAD-AutoLISP-Reference/files/GUID-16FB5411-7563-4FB9-AB59-A73457B092C5.htm)

**Supported Platforms:**  Windows, Mac OS, and Web

## Signature

```lisp
(terpri)
```

No arguments.

## Return Values

**Type:**  nil

Always returns `nil`.

## Remarks

The `terpri`  function is not used for file I/O. To write a newline to a file, use `prin1`, `princ`, or `print`.
